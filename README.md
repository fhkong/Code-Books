# 一些读过和将要读的书籍 :smile:

**写在前面**  
个人感觉大学里面学到的知识都是入门级，要想真正深入一个领域时，仅仅靠大学课上学到的知识是完全不够的，必须要利用课余的时间来大量阅读一些相关的书籍。

关于看书还是看一些技术博客  
想要了解一些知识，看一些大牛的技术博客是有帮助的。  
要想深入一个领域，看一些书籍才能真正的学习到本领。  

关于看电子书还是看纸质书  
我个人感觉还是偏向于看纸质书，但也不厌烦看电子PDF  


还有就是看书一定要有耐心，耐住自己的性子。
